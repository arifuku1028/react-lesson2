# React の環境構築

## 環境説明

- Node.js v14.17.4
- Yarn v1.22.18
- React/React-DOM v17.0.2
- ESLint, Prettier（保存時に実行・整形）

## 実行方法

1. GitHub リポジトリから Clone
   ```
   git clone https://github.com/arifuku1028/react-lesson2.git ./my-app
   ```
2. カレントディレクトリの移動
   ```
   cd my-app
   ```
3. React 実行
   ```
   yarn start
   ```
4. 実行確認  
   http://localhost:3000
